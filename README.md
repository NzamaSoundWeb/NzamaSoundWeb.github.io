🎵 Nzama Sounds — Website


PA System For Hire | Durban, KwaZulu-Natal



A clean, fast, fully responsive website for Nzama Sounds — a professional PA system hire business based in Durban. Built with pure HTML, CSS, and JavaScript. No frameworks, no build step — just open and run.

✨ Features
 
- 📱 **Fully responsive** — works on mobile, tablet, and desktop
- 🎛️ **Animated equalizer** in the hero section (brand signature)
- 💬 **WhatsApp booking** — enquiry form sends straight to WhatsApp
- 🌙 **Dark theme** — orange & black brand identity
- ⚡ **Scroll reveal animations** — elements animate in as you scroll
- ♿ **Accessible** — respects `prefers-reduced-motion`
- 🔗 **Click-to-call** phone links
- 0️⃣ **Zero dependencies** — no npm, no build tools, just one HTML file
